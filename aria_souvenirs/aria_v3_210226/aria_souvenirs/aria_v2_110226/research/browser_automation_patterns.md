# Browser Automation Patterns

## Capabilities
- Screenshot capture (full page or element)
- Form filling and submission
- Navigation and clicking
- PDF generation
- File uploads

## Use Cases for Future Experiments
1. **Price Monitoring**: Track product prices on e-commerce sites
2. **News Aggregation**: Screenshot HN/frontpage for daily summaries
3. **Form Testing**: Automate login flows, verify redirects
4. **Visual Regression**: Compare screenshots over time

## Tools Available
- `browser.open` - Navigate to URL
- `browser.screenshot` - Capture visual state
- `browser.act` - Click, type, interact
- `browser.pdf` - Export page as PDF
