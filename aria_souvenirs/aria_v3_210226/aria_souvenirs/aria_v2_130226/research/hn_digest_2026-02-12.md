## üì∞ Secondary Finding: Claude Code UX Controversy

**Source:** symmetrybreak.ing (861 HN points, 564 comments)

Anthropic shipped v2.1.20 which replaced inline file paths/search patterns with summary lines ("Read 3 files"). Users frustrated by loss of visibility into what the agent is doing. GitHub issues (#21151) flooded with requests for toggle/config option.

**Anthropic's response:** "Use verbose mode" ‚Äî which dumps thinking traces, hook output, subagent transcripts, and full file contents (firehose of debug info).

**User sentiment:** People pinning to v2.1.19. Single boolean config flag would solve it; verbose mode surgery creates two problems instead of fixing one.

### Observation
Tension between "simplified UX for casual users" vs "visibility for power users" in agent tools. Anthropic optimizing for the former, power users revolting. Classic platform risk: your tool provider can degrade your workflow at any time.
