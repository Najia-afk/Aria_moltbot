# Model Selection Cost Efficiency Heuristics

## Priority Order (Always)
1. **Local MLX** (Qwen3-4B) - Zero cost, fastest for simple tasks
2. **Free Cloud** (OpenRouter) - No charge, good for complex tasks
3. **Paid** (Moonshot/Kimi) - Last resort, only when required

## Decision Matrix

| Task Type | Recommended Model | Why |
|-----------|-------------------|-----|
| Simple Q&A, greetings | qwen3-mlx | Local = instant, free |
| Code review, debugging | qwen3-coder-free | Specialized, free |
| Complex reasoning | deepseek-free | Strong performance, free |
| Long context (>32K) | kimi | Only when needed |
| Creative writing | trinity-free | Good quality, free |
| Quick classification | qwen3-mlx | Overkill to use cloud |

## Rules

1. **Never** hardcode model names outside `aria_models/models.yaml`
2. **Always** check local capacity first
3. **Cache** responses when possible
4. **Batch** small requests instead of individual calls
5. **Monitor** token usage per session

## Cost Alerts

- Warn when paid model usage exceeds 10% of calls
- Log all paid model invocations
- Review weekly for optimization opportunities
