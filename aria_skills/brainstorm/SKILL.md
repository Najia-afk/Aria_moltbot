# aria-brainstorm

Creative ideation and brainstorming session management.

## Purpose
- Start structured brainstorming sessions.
- Capture, connect, and evaluate ideas.
- Apply ideation techniques and summarize outcomes.

## Main Tools
- `start_session`
- `add_idea`
- `apply_technique`
- `connect_ideas`
- `evaluate_ideas`
- `summarize_session`
