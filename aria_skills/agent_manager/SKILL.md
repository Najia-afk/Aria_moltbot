# aria-agent-manager

Runtime agent lifecycle management for Aria.

## Purpose
- Spawn, list, monitor, and terminate managed agent sessions.
- Produce performance summaries and prune stale sessions.

## Main Tools
- `list_agents`
- `spawn_agent`
- `terminate_agent`
- `get_agent_stats`
- `prune_stale_sessions`
- `get_performance_report`
