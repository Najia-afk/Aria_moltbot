# aria-working-memory

Persistent short-term memory operations for Aria runtime context.

## Purpose
- Store, retrieve, rank, checkpoint, and prune working memory.
- Provide high-relevance context for decision and response flows.

## Main Tools
- `remember`
- `recall`
- `get_context`
- `checkpoint`
- `restore_checkpoint`
- `forget`
- `update`
- `reflect`
- `sync_to_files`
